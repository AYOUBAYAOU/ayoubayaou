ماهي مكتبة react؟


مكتبة React هي مكتبة برمجية للجافاسكريبت تم تطويرها من قبل فيسبوك وتستخدم لتطوير واجهات المستخدم UI لتطبيقات الويب. تتميز React بأنها تعتمد على مفهوم المكونات Components، حيث يتم تقسيم واجهة المستخدم إلى عدة مكونات صغيرة يتم تجميعها لتكوين صفحة ويب بأكملها.
وتعتبر React من أشهر المكتبات المستخدمة لتطوير واجهات المستخدم حالياً، ويستخدمها العديد من الشركات الكبيرة مثل فيسبوك وإنستغرام وأمازون وغيرها، وذلك بسبب سهولة استخدامها وقدرتها على تحسين أداء تطبيقات الويب. كما تتوافر لـ React مجموعة كبيرة من المكتبات والأدوات التي تساعد على تطوير تطبيقات الويب بشكل أسرع وأكثر كفاءة.

تنزيل مكتبة react


لتنزيل مكتبة React وإستخدامها يجب عليك إتباع مجموعة من الخطوات:
تثبيت Node.js و npm : React يعتمد على npm لإدارة التبعيات dependencies وتثبيتها، لذلك يجب تثبيت Node.js و npm أولاً.
إنشاء تطبيق React: بعد تثبيت Node.js و npm يمكن إنشاء تطبيق React بسهولة باستخدام create-react-app. يمكن القيام بذلك بتنفيذ الأمر التالي في وحدة التحكم command line:

cmd 
C:\Users npx create-react-app my-app

وسيقوم هذا الأمر بإنشاء تطبيق React جديد في مجلد my-app.
تثبيت المكتبة: لتثبيت المكتبة React بشكل منفصل، يمكن تنفيذ الأمر التالي في وحدة التحكم:

cmd 
C:\Users npm install react

أو يمكن تثبيت React مع ReactDOM التي تستخدم لربط React مع DOM باستخدام الأمر التالي:

cmd 
C:\Users npm install react-dom

ولتشغيل مشروع react ندهب إلى مسار المشروع وندخل الأمر الثالي:

cmd 
C:\Users\my-app npm run serve

بعد تنفيذ الخطوات السابقة، يمكن البدء في استخدام React في تطبيقات الويب الخاصة بك.
