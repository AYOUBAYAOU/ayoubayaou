أفضل التطبيقات للهاتف للبرمجة


في هذا المقال سنتعرف على تطبيقات للبرمجة من الهاتف توجد في جوجل يمكنك البحث عنها سأترك لك الأسماء هنا ويمكنك البحث عنها وتجربتها إلى حين تشتري حاسوب إن شاء الله هنا في هذا المقال تحدثنا عن أفضل التطبيقات للحاسوب يمكنك تفقده والآن أترككم مع التجميع لأفضل التطبقات لتعلم البرمجة من الهاتف:

  • spck editor
    • محرر رائع مخصص للويب بالتحديد يدعم لغات html css javascript

  • CxxDroid
    • تريد كتابة أكواد لغة C++ أنصحك بهذا التطبيق الرائع أفضل تطبيق لدي لكتابة أكواد C++ وربما يدعم حتى لغة C

  • pydriod 3
    • هذا التطبيق مشهور جدا حرفيا هو أفضل تطبيق لكتابة أكواد لغة python نعم توجد محررات أخرى ولكن هذا أفضلهم.

  • Aide
    • قد تكون تريد دخول مجال التطبيقات بلغة Java and kotlin ولا تملك حاسوب هذا التطبيق هو منقدك يدعم تطوير التطبيقات بلغات java and kotlin

  • its Magic
    • قد تكون تريد دخول مجال تطوير الألعاب تلاتية الأبعاد ولا تملك حاسوب هذا المحرك سينفعك إلى حين شراء حاسوب.

  • max2d
    • تريد تطوير الألعاب ثنائية الأبعاد ولا تملك حاسوب هذا التطبيق سيأدي الغرض إلى حين تشتري حاسوب.


لا توجد لك أعدار الأن لعدم البدء لو لم يكن لذيك حاسوب هذه قائمة تطبيقات من مختلف المجالات ماذا تنتطر و وفقك الله.
