ماهو Type Coercion؟

يوجد شيء إسمه type coercion يعني أن البيانات تتحول بشكل تلقائي هنا لذينا الأرقام كلها عبارة عن string وليس رقم مثلا لذينا: 

console.log("4" + "4");
ماهو الناتج هل سيكون 8 لا سيكون 44 لماذا؟ لأن الجافا سكريبت بإستعمال علامة + تعمل concatenation بين two String مثلا لو كتبت هكذا: 

console.log("Ayoub" + "Ayaou");
// output "Ayoub Ayaou"
مثل هذا حدث مع الأرقام والآن لنفترض إستعملنا علامة غير + يعني أحد هذه العلامات (* - /) ماذا سيحدث مثال 
console.log("2" - "1");
// output 1
ماذا سيكون الناتج سيكون 1 لماذا هنا لدينا string وليس رقم إذا كيف تمت العملية الحسابية هنا يأتي دور Type Coercion تحويل تلقائي للبيانات بدون تدخلك عند إستعمال علامات غير + ستتحول من String إلى رقم يعني لو كتبنا بهذه الطريقة 
console.log("Ayoub" * "Ayaou");
// output  NaN
كما ترون ولكن بإستخدام + ستحصل عملية concatenation والآن نأتي إلى 
console.log("2" + "2");
// output  "22"
حدثث عملية concatenation والآن 
console.log("22" - "2" );
// output  20
لماذا لأن "2" التي هي string تحول نوعها إلى number هي و "22" و حدثث العملية الحسابية والآن إنتهينا من مقالتنا للفهم أكثر شرحنا هذا في قناتنا في دورة لغة الجافا سكريبت يمكنك تفقده وشكرا لكم.
