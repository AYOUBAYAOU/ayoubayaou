ماهي لغة CSS ؟

تعتبر لغة ال css من أهم اللغات التي يجب عليك تعلمها في مسار تطوير المواقع لكي تصبح مصمم ويب محترف يجب عليك تعلم هذه اللغة لماذا ؟ لأن وظيفتها هي تنسيق صفحات الويب طبعا تحتاج إلى معرف لغة html للدخول إلى هذه اللغة هذا مقال عن لغة html إعتبر ال CSS مثل ديكور الخاص بالبيت قلنا سابقا أنا لغة ال html هي هيكل المواقع و مثلناها بالبيت هي عبارة عن أساس البيت بعد الإنتهاء من أساس البيت مالذي يلي، يلي الصباغة و الديكور لكي يصبح جداب ويسكن فيه الناس أو يزورنه إعتبر CSS هكذا وكلمة CSS هي إختصار لكلمة Cascading style sheet وهي ليست لغة برمجة مما يحيل على سهولة هذه اللغة هي عبارة عن لغة تصميم كما سترى الآن في الكود لكتابة كود ال css نحتاج لكتابة المحدد وهو ال selector يعني نحدد هنا إسم وسم أو باللغة الإنجليزية tag ال html مثلا لدينا وسم h1 سوف نقوم بتحديده هناك في مكان ال selector ومن ثم نعمل أقواس المعقوفة هنا داخل الأقواس المعقوفة نكتب كل التنسيقات الخاصة بالوسم مثل اللون العرض والطول و الحجم وغيرها... نكتب إسم الخاصية و نكتب هناك القيمة الخاصة بها كما يظهر معكم في الكود: 

selector {
    property-name: value;
}
في كل مرة نكتب الخاصة مع القيمة نكتب نقطة فاصلة للتفرقة بين كل الخصائص هنا كود حقيقي للغة كما ترى نكتب المحدد هنا وسم ال h1 ونعمل الأقواس المعقوفة داخلها نكتب الخاصة للتحكم مثلا باللون والقيمة نطع اللون الذي سيكون لون النص أحمر وأعيدها طروري ذائما عمل نقطة فاصلة في آخر كل خاصية عند كتابة أكثر من خاصية كما ظاهر معكم في الكود هنا: 

h1 {
    color: red;
}
هنا عند كتابة أكثر من خاصية: 

selector {
    property-name: value;
    property-name: value;
    property-name: value
}
تنبيه مثلا عند كتابة إسم مركب مثل لون الخلفية نستعمل علامة الداش للتفرقة بينهم كما هو ظاهر هنا: 

div {
    background-color: red;
}
وهكذا إنتهينا من سلسلة الأكود, مرت لغة ال CSS كذلك من عدة إصدارات وصولا إلى آخر إصدار وهو الثالث حيث أضيف فيه العديد من المزايا كما ترون وكان أول إصدار عام 1996
